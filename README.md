# JH'S 웹 포털 웹앱 안내

JH'S 웹 포털은 수업 운영과 학생 학습을 돕는 두 가지 웹앱으로 연결됩니다.

- JH'S 스크린: https://screen-lemon.vercel.app/
- 초등 개념어 마스터: https://concept-kappa-khaki.vercel.app/

## JH'S 스크린

### 1. 목적

JH'S 스크린은 교실 전면 화면이나 교사용 기기에서 수업 진행에 필요한 정보를 한곳에 띄워 두기 위한 교사용 대시보드 웹앱입니다.

오늘의 시간표, 안내 사항, 현재 날짜와 시계를 한 화면에 배치하여 학생들이 수업 흐름을 쉽게 확인할 수 있도록 돕습니다. 또한 타이머, 발표자 랜덤 뽑기, 우리반 명단 관리, 교사용 사이트 바로가기 기능을 제공하여 수업 중 반복적으로 사용하는 도구를 빠르게 실행할 수 있습니다.

### 2. 화면구성

- 오늘의 시간표: 교시와 과목을 입력하고 수정할 수 있는 시간표 영역입니다.
- 안내 사항: 굵게, 기울임, 밑줄, 글씨 크기, 글씨 색상을 조절할 수 있는 안내문 작성 영역입니다.
- 현재 시간: 날짜, 아날로그 시계, 디지털 시계를 함께 보여주는 시간 확인 영역입니다.
- 우측 상단 바로가기: 유튜브, 인디스쿨, 아이스크림, T셀파, 비바샘 등 교사용 사이트로 이동하는 버튼입니다.
- 명단 관리: 학생 이름을 쉼표 또는 줄바꿈으로 입력해 우리반 명단을 저장하는 모달입니다.
- 플로팅 도구: 배경 화면 바꾸기, 타이머, 랜덤 발표 뽑기를 실행하는 하단 도구 버튼입니다.

### 3. 사용방법

1. 웹 포털에서 `JH'S 스크린` 버튼을 클릭합니다.
2. `오늘의 시간표`에 교시와 과목을 입력하고, 필요한 경우 추가 또는 삭제합니다.
3. `안내 사항` 영역에 학생들에게 보여줄 할 일, 준비물, 수업 안내 등을 입력합니다.
4. 우측 상단의 `명단 관리`를 눌러 학생 이름을 저장합니다.
5. 수업 중 발표자가 필요하면 하단의 랜덤 뽑기 도구를 열고 `뽑기 시작`을 누릅니다.
6. 활동 시간을 관리할 때는 하단의 타이머 도구를 열어 시간을 설정한 뒤 시작합니다.
7. 필요에 따라 배경 화면을 바꾸거나 교사용 사이트 바로가기 버튼을 사용합니다.

## 초등 개념어 마스터

### 1. 목적

초등 개념어 마스터는 초등 3~6학년 학생들이 국어, 사회, 과학 교과의 핵심 개념어를 단계적으로 익히도록 돕는 학습 웹앱입니다.

개념어의 뜻, 쉬운 설명, 예시 문장, 한자어, 비슷한말, 반대말, 관련어 등을 함께 제공하여 단어의 의미를 깊이 이해하도록 구성되어 있습니다. 예비 테스트, 단계별 학습, 개념어 놀이터, 마지막 확인 테스트를 통해 학습 전 진단부터 복습과 확인까지 이어지도록 돕습니다.

### 2. 화면구성

- 학년 선택 화면: 초등 3학년, 4학년, 5학년, 6학년 중 학습할 학년을 선택합니다.
- 과목 선택 화면: 국어, 사회, 과학 중 학습할 과목을 선택합니다.
- 시작 화면: 선택한 학년과 과목의 전체 개념어 수를 보여주고, 예비 테스트 또는 바로 학습을 선택합니다.
- 예비 테스트: 전체 개념어 중 20문항을 뽑아 OX와 객관식 문제로 사전 이해도를 확인합니다.
- 단계별 학습: 개념어를 20개씩 나누어 1~4단계로 학습합니다.
- 개념어 카드: 뜻, 쉽게 풀이한 설명, 예시, 한자어, 비슷한말, 반대말, 관련어, 학습 팁 등을 보여줍니다.
- 개념어 놀이터: 카드 매칭과 OX 스피드 게임으로 개념어를 복습합니다.
- 마지막 확인 테스트: 학습 후 20문항으로 최종 이해도를 확인합니다.

### 3. 사용방법

1. 웹 포털에서 `초등 개념어 마스터` 버튼을 클릭합니다.
2. 첫 화면에서 학습할 학년을 선택합니다.
3. 국어, 사회, 과학 중 학습할 과목을 선택합니다.
4. 학습 전 수준을 확인하고 싶으면 `예비 테스트`를 선택하고, 바로 공부하려면 `바로 학습`을 선택합니다.
5. 단계별 학습에서 개념어 카드의 뜻, 예시, 한자어, 관련어를 차례로 읽습니다.
6. 한 단계의 개념어를 모두 확인하면 다음 단계로 이동합니다.
7. 4단계까지 학습한 뒤 `개념어 놀이터`에서 카드 매칭 또는 OX 스피드 게임으로 복습합니다.
8. 학습을 마치면 `마지막 확인 테스트`를 풀고 결과를 확인합니다.
9. 결과에서 헷갈린 개념이 있으면 다시 학습으로 돌아가 복습합니다.
